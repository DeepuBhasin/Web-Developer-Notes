1. Hosts Files is a plain test operating system file that maps domain names to IP addresses. 
2. Hosts files serves the function of translating human-friendly hostnames into numeric Internet Protocol(IP) addresses, to identify and locate a host in an ip network like DNS
3. A hosts file can be edited to override your network settings and thus allow your local machine to view a website prior to the DNS changes propagating.
4. You Can do 
    a. Assign Local domains name 
    b. Redirect websites 
    c. Blocking websites 
    d. Create websites Shortcuts 

5. Locations 

    Window : c/Windows/system32/drivers/etc/hosts.conf
    linux  : /etc/hosts
